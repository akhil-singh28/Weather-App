# Weather-App
# 🌦️ Weather App

A simple Weather App that fetches real-time weather data using an API.

## 🚀 Features
- Displays current weather conditions
- Shows temperature, humidity, and wind speed
- Responsive and easy-to-use interface

## 🔧 Tech Stack
- HTML, CSS, JavaScript
- OpenWeather API (or any other weather API)

## 📦 Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/akhil-singh28/Weather-App.git
Navigate to the project folder:


->cd Weather-App
Install dependencies (if any):

->npm install

🔗 API Integration
This app fetches weather data using an API. You need an API key from OpenWeather:

Sign up at OpenWeather

Generate your API key

Update the API key in the JavaScript file:
->const apiKey = 'YOUR_API_KEY';
🛠 Usage
Run the app in your browser:

Open index.html in a browser

Enter the city name and get real-time weather updates!
